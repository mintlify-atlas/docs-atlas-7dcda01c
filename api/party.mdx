---
title: Party
description: Represents a party (company or person) in an invoice
---

The `Party` class represents either the seller or buyer in a FacturaE invoice. It supports both legal entities (companies) and natural persons.

## Static Constructors

### company()

Create a legal entity (company) party.

<ParamField path="taxNumber" type="string" required>
  Tax identification number (NIF/CIF)
</ParamField>

<ParamField path="name" type="string" required>
  Company legal name
</ParamField>

<ResponseField name="return" type="Party">
  Returns a new Party instance representing a company
</ResponseField>

```php
$company = Party::company('B12345678', 'Mi Empresa SL');
```

### person()

Create a natural person party.

<ParamField path="taxNumber" type="string" required>
  Tax identification number (NIF/DNI)
</ParamField>

<ParamField path="name" type="string" required>
  Person's first name
</ParamField>

<ParamField path="firstSurname" type="string" required>
  Person's first surname
</ParamField>

<ParamField path="lastSurname" type="string|null">
  Person's second surname (optional)
</ParamField>

<ResponseField name="return" type="Party">
  Returns a new Party instance representing a person
</ResponseField>

```php
$person = Party::person(
    taxNumber: '12345678A',
    name: 'Juan',
    firstSurname: 'García',
    lastSurname: 'López'
);
```

## Fluent Setters

### address()

Set the party's address.

<ParamField path="street" type="string" required>
  Street address
</ParamField>

<ParamField path="postalCode" type="string" required>
  Postal code
</ParamField>

<ParamField path="town" type="string" required>
  Town/city name
</ParamField>

<ParamField path="province" type="string" required>
  Province name
</ParamField>

<ParamField path="countryCode" type="string" default="ESP">
  ISO 3166-1 alpha-3 country code
</ParamField>

<ResponseField name="return" type="Party">
  Returns self for method chaining
</ResponseField>

```php
$party->address(
    street: 'Calle Mayor 1, 3º A',
    postalCode: '28001',
    town: 'Madrid',
    province: 'Madrid',
    countryCode: 'ESP'
);
```

<ParamField path="name" type="string|null">

  Returns self for method chaining
</ResponseField>

## Complete Example

```php
use PhpFacturae\Party;

// Company example
$seller = Party::company('B12345678', 'Mi Empresa SL')
    ->address(
        street: 'Calle Mayor 1, 3º A',
        postalCode: '28001',
        town: 'Madrid',
        province: 'Madrid'
    )
    ->tradeName('Nombre Comercial')
    ->email('ventas@empresa.com')
    ->phone('+34912345678')
    ->website('https://www.empresa.com')
    ->cnoCnae('6201');

// Person example
$buyer = Party::person(
    taxNumber: '12345678A',
    name: 'Juan',
    firstSurname: 'García',
    lastSurname: 'López'
)
    ->address(
        street: 'Gran Vía 10',
        postalCode: '28013',
        town: 'Madrid',
        province: 'Madrid'
    )
    ->email('juan.garcia@example.com')
    ->phone('+34987654321');
```
