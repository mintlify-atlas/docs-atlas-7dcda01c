title: Invoice
description: Main class for creating and managing FacturaE invoices
---

The `Invoice` class is the core component of PHP FacturaE. It provides a fluent interface for building electronic invoices compliant with the Spanish FacturaE format.

## Creating an Invoice

### create()

Create a new invoice instance.

<ParamField path="number" type="string" required>
  The invoice number
</ParamField>

<ResponseField name="return" type="Invoice">
  Returns a new Invoice instance
</ResponseField>

```php
$invoice = Invoice::create('INV-2024-001');
```

## Invoice Lines

### line()

Add a standard invoice line with Spanish tax shortcuts.

<ParamField path="description" type="string" required>
  Line item description
</ParamField>

<ParamField path="price" type="float" required>
  Unit price (without taxes)
</ParamField>

<ParamField path="quantity" type="float" default="1">
  Quantity
</ParamField>

<ParamField path="vat" type="float|null">
  VAT (IVA) rate percentage
</ParamField>

<ParamField path="irpf" type="float|null">
  Income tax withholding (IRPF) rate percentage
</ParamField>

<ParamField path="igic" type="float|null">
  Canary Islands tax (IGIC) rate percentage
</ParamField>

<ParamField path="surcharge" type="float|null">
  Equivalence surcharge rate percentage
</ParamField>

<ParamField path="ie" type="float|null">
  Special tax (IE) rate percentage
</ParamField>

<ParamField path="ieWithheld" type="bool" default="false">
  Whether special tax is withheld
</ParamField>

<ParamField path="discount" type="float|null">
  Line discount amount
</ParamField>

<ParamField path="articleCode" type="string|null">
  Product/article code
</ParamField>

<ParamField path="detailedDescription" type="string|null">
  Additional detailed description
</ParamField>

<ParamField path="unit" type="UnitOfMeasure|null">
  Unit of measure enum value
</ParamField>

<ResponseField name="return" type="Invoice">
  Returns self for method chaining
</ResponseField>

```php
// Standard line with VAT
$invoice->line('Lámpara LED', quantity: 3, price: 20.14, vat: 21);

// Service with VAT and IRPF withholding
$invoice->line('Consultoría', price: 500, vat: 21, irpf: 15);

// Canary Islands product with IGIC
$invoice->line('Producto Canarias', price: 100, igic: 7);

// Jewelry with equivalence surcharge
$invoice->line('Joyería', price: 200, vat: 21, surcharge: 5.2);

// Electricity with custom unit
$invoice->line('Electricidad', price: 80, vat: 21, unit: UnitOfMeasure::KWh);
```

### customLine()

Add a line with custom tax configuration.

<ParamField path="description" type="string" required>
  Line item description
</ParamField>

<ParamField path="price" type="float" required>
  Unit price
</ParamField>

<ParamField path="taxes" type="TaxBreakdown[]" required>
  Array of TaxBreakdown objects
</ParamField>

<ParamField path="quantity" type="float" default="1">
  Quantity
</ParamField>

<ParamField path="discount" type="float|null">
  Line discount amount
</ParamField>

<ParamField path="articleCode" type="string|null">
  Product/article code
</ParamField>

<ParamField path="unit" type="UnitOfMeasure|null">
  Unit of measure enum value
</ParamField>

<ParamField path="specialTaxableEvent" type="SpecialTaxableEvent|null">
  Special taxable event enum value
</ParamField>

<ParamField path="specialTaxableEventReason" type="string|null">
  Reason for special taxable event
</ParamField>

<ResponseField name="return" type="Invoice">
  Returns self for method chaining
</ResponseField>

```php
use PhpFacturae\Entities\TaxBreakdown;
use PhpFacturae\Enums\Tax;

$taxes = [
    new TaxBreakdown(Tax::IVA, 21),
    new TaxBreakdown(Tax::IRPF, 15),
];

$invoice->customLine('Servicio profesional', price: 1000, taxes: $taxes);
```

### payment()

Add a custom payment object.

<ParamField path="payment" type="Payment" required>
  Payment entity instance
</ParamField>

<ResponseField name="return" type="Invoice">

</ResponseField>

use PhpFacturae\Entities\Payment;

### splitPayments()

Split payment into multiple installments.

<ParamField path="method" type="PaymentMethod" required>
  Payment method enum value
</ParamField>

<ParamField path="installments" type="int" required>
  Number of installments
</ParamField>

<ParamField path="firstDueDate" type="string" required>
  Due date for the first installment
</ParamField>

<ParamField path="intervalDays" type="int" default="30">
  Days between each installment
</ParamField>

<ParamField path="iban" type="string|null">
  Bank account IBAN (required for Transfer and DirectDebit)
</ParamField>

<ResponseField name="return" type="Invoice">
  Returns self for method chaining
</ResponseField>

```php
use PhpFacturae\Enums\PaymentMethod;

$invoice->splitPayments(
    method: PaymentMethod::Transfer,
    installments: 3,
    firstDueDate: '2024-02-15',
    intervalDays: 30,
    iban: 'ES7921000813610123456789'
);
```

## Corrective Invoices

### corrects()

Mark this invoice as a correction of another invoice.

<ParamField path="invoiceNumber" type="string" required>
  Number of the invoice being corrected
</ParamField>

<ParamField path="reason" type="CorrectionReason" default="CorrectionReason::TransactionDetail">
  Reason for the correction
</ParamField>

<ParamField path="method" type="CorrectionMethod" default="CorrectionMethod::FullReplacement">
  Correction method
</ParamField>

<ParamField path="series" type="string|null">
  Series of the invoice being corrected
</ParamField>

<ParamField path="periodStart" type="string|DateTimeImmutable|null">
  Start of correction period
</ParamField>

<ParamField path="periodEnd" type="string|DateTimeImmutable|null">
  End of correction period
</ParamField>

<ResponseField name="return" type="Invoice">
  Returns self for method chaining
</ResponseField>

```php
use PhpFacturae\Enums\CorrectionReason;
use PhpFacturae\Enums\CorrectionMethod;

$invoice->corrects(
    invoiceNumber: 'INV-2024-001',
    reason: CorrectionReason::TransactionDetail,
    method: CorrectionMethod::FullReplacement
);
```

## Output

### toXml()

Generate the FacturaE XML string.

<ResponseField name="return" type="string">
  XML string of the invoice
</ResponseField>

<ResponseField name="throws" type="InvoiceValidationException">
  If the invoice fails validation
</ResponseField>

```php
try {
    $xml = $invoice->toXml();
    echo $xml;
} catch (InvoiceValidationException $e) {
    print_r($e->getErrors());
}
```

<ParamField path="path" type="string" required>

</ParamField>

<ResponseField name="return" type="Invoice">
  Returns self for method chaining
</ResponseField>

<ResponseField name="throws" type="InvoiceValidationException">
  If the invoice fails validation
</ResponseField>

```php

```
